115 /*
116  * @brief cuDNN implementation of ConvolutionLayer.
117  * Fallback to ConvolutionLayer for CPU mode.
118  *
119  * cuDNN accelerates convolution through forward kernels for filtering and bias
120  * plus backward kernels for the gradient w.r.t. the filters, biases, and
121  * inputs. Caffe + cuDNN further speeds up the computation through forward
122  * parallelism across groups and backward parallelism across gradients.
123  *
124  * The CUDNN engine does not have memory overhead for matrix buffers. For many
125  * input and filter regimes the CUDNN engine is faster than the CAFFE engine,
126  * but for fully-convolutional models and large inputs the CAFFE engine can be
127  * faster as long as it fits in memory.
128 */
